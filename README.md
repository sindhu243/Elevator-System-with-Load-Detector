# Elevator System with Load Detector

## Overview
This project demonstrates an Arduino-based Elevator System with Load Detection. The system monitors the elevator load using a load sensor and prevents movement when the weight exceeds the safe limit.

## Features
- Load detection using HX711
- Overload warning with buzzer
- Automatic elevator control
- Floor selection
- Safe operation
- Low-cost prototype

## Hardware Used
- Arduino Nano
- HX711 Load Cell Module
- Load Cell
- L298N Motor Driver
- DC Gear Motor
- Servo Motor
- LCD Display
- Buzzer
- Limit Switches
- Power Supply
